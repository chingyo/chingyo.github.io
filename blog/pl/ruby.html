Compile and install Ruby on Ubuntu

Ok, I use the Ubuntu 18.04.5 LTS.

First, you must install some libs:
sudo apt install autoconf bison libssl-dev libyaml-dev libreadline-dev zlib1g-dev libncurses5-dev libffi-dev libgdbm-dev libdb-dev

And then download source code from: ruby-lang.org

Finally, running the command:
./configure [--prefix=/home/username/ruby]
make
[sudo] make install
